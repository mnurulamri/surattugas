# Aplikasi Surat Tugas Online

Sistem pengajuan surat tugas berbasis web yang digunakan untuk mendukung proses administrasi surat tugas secara digital di lingkungan FISIP UI.

---

## Deskripsi

Aplikasi Surat Tugas Online dikembangkan untuk menggantikan proses pengajuan surat tugas manual berbasis dokumen fisik menjadi sistem digital berbasis web yang lebih efisien, terdokumentasi, dan mudah dimonitor.

Sistem digunakan oleh seluruh unit di lingkungan FISIP UI untuk:
- pengajuan surat tugas,
- monitoring status pengajuan,
- proses approval,
- dan dokumentasi administrasi surat tugas.

---

## Latar Belakang

Sebelum sistem dikembangkan, proses pengajuan surat tugas dilakukan secara manual menggunakan dokumen kertas sehingga menyebabkan:

- penggunaan kertas cukup tinggi,
- proses disposisi membutuhkan waktu lama,
- monitoring pengajuan sulit dilakukan,
- dan arsip administrasi belum terdigitalisasi dengan baik.

Untuk mendukung transformasi digital administrasi, dikembangkan aplikasi berbasis web yang memungkinkan seluruh proses dilakukan secara online.

---

## Tujuan Pengembangan

- Mengurangi penggunaan dokumen fisik dan kertas
- Mendukung digitalisasi layanan administrasi
- Mempercepat proses pengajuan dan approval
- Mempermudah monitoring pengajuan surat tugas
- Menyediakan dokumentasi administrasi yang lebih terstruktur
